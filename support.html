アプリ名：冷笑ダメ絶対（Reishou Dame Zettai）

キーボードの設定方法

  1. 「冷笑ダメ絶対」アプリを一度開く
  2. 「設定」アプリ → 一般 → キーボード → キーボード → 新しいキーボードを追加
  3. リストから「冷笑ダメ絶対」（フリック入力）または「冷笑ダメ絶対 ローマ字」を選んで追加
  4. 追加したキーボードをタップ →「フルアクセスを許可」をオン

フルアクセスは振動・通知のためだけに使用します。入力内容の収集や外部送信は一切行いません。

よくある質問

Q. キーボードが切り替え候補に出てこない
A. 一度キーボードを削除して、上記の手順で追加し直してみてください。iOSの設定反映に時間がかかる場合があります。

Q. 振動しない / 通知が来ない
A. 「フルアクセスを許可」がオンになっているか確認してください（設定 → 一般 → キーボード → キーボード → 該当キーボード）。また、本体の「通知」設定でこのアプリの通知が許可されているかもご確認ください。

Q. 変換候補が出ない
A. ひらがなを入力すると自動で候補バーに表示されます。候補をタップすると確定します。

Q. データは外部に送信されますか？
A. いいえ。入力内容の検知・変換はすべて端末内で完結しており、外部サーバーへの送信は一切行っていません。詳しくはプライバシーポリシーをご覧ください。

Support (English)

How to enable the keyboard

  1. Open the "冷笑ダメ絶対" app once
  2. Settings → General → Keyboard → Keyboards → Add New Keyboard
  3. Select "冷笑ダメ絶対" (flick) or "冷笑ダメ絶対 ローマ字" (romaji)
  4. Tap the added keyboard → enable "Allow Full Access"

Full Access is used only for vibration and local notifications. We never collect or transmit your typed content.

FAQ

The keyboard doesn't work / vibrate / notify?
Make sure "Allow Full Access" is enabled for the keyboard, and notifications are allowed for this app in system Settings.
